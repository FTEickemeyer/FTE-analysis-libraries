Spectrum
========

Subclasses of ``xy_data`` for optical spectra. Loads Andor, OceanOptics,
and Avantes files. Includes spectral math: photon-flux conversion, EQE
calculation, blackbody-tail fitting, PLQY parameter extraction, and
Urbach-energy analysis.
