IV
==

``IVData`` class for current-density/voltage measurements. Loads Biologic,
Ossila, and Zahner JV files. Extracts Voc, Jsc, FF, PCE; fits the
single-diode model; computes Shockley–Queisser and radiative limits; and
produces loss-analysis plots.
