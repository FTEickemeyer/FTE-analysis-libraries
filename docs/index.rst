FTE Analysis Libraries
======================

Scientific Python library for photovoltaic and energy-materials analysis
(EPFL / FTE group). Covers data import, processing, fitting, and
visualisation for JV curves, EIS, PL spectroscopy, TRPL, PLQY,
redox-flow batteries, and related techniques.
